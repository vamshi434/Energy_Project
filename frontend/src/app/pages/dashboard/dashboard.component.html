Technical Overview

This application is a Single Page Application (SPA) with a decoupled frontend and backend architecture.

  • Frontend: Built with Angular, it communicates with the backend via HTTP calls. The application uses JWT for authentication, ensuring secure access to protected routes. The UI is designed with accessibility in mind, adhering to ADA/WCAG principles.
  • Backend: Developed using Node.js and Express, it handles authentication and serves data for dynamic charts. JWT tokens are used to validate requests, maintaining secure communication between the frontend and backend.
  • Deployment: Both frontend and backend are hosted on the same server. NGINX serves the frontend on port 80, while the backend runs on port 3000. This setup ensures efficient routing and resource management.

The application is structured to provide a seamless user experience while maintaining a clear separation of concerns between the frontend and backend components.
